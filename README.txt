Stay Fit  a simple application that motivates users to excercise making it seem simple and easy. It was meant to be used as a public installation that uses kinect to detect user jogging and encouraging them to continue doing so to reveal a surprise. You can view the project demo at following link : https://www.youtube.com/watch?v=SUl9jHWphSY

1. Connect Kinect v1.8 to the laptop
2. Run the stay-fit.py application and stay-fit.pde processing application
3. Jog infront the application to unveil the animation
